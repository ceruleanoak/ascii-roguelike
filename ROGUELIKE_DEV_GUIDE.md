# Roguelike Development Guide

This project has a specialized development skill available at `~/.claude/SKILL_ROGUELIKE_DEV.md`

## Quick Usage

When working on this roguelike, you can request:

```
/rl validate-all              - Full validation of game balance
/rl enemy Spider mid          - Generate new mid-tier enemy
/rl weapon laser advanced     - Generate advanced weapon
/rl validate-recipes          - Check all recipes for conflicts
/rl balance-enemies           - Analyze enemy difficulty curve
/rl missing-recipes           - Find unused ingredient combinations
```

## What The Skill Provides

1. **Quick Entity Generation** - Balanced enemies, weapons, items, recipes
2. **Validation Tools** - Check for conflicts, balance issues, integration problems
3. **Balance Analysis** - Enemy curves, weapon progression, drop rates
4. **Testing Workflows** - Simulate drops, test recipes, verify builds
5. **Templates** - Complete feature implementations
